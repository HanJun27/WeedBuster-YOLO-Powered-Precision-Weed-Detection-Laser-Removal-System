# ASR 语音指令集

> 适用版本: v3.14.0+ | 更新日期: 2026-07-12

---

## 概述

系统支持 **9 条标准指令**，共 **40+ 触发词**，通过端侧 ASR 引擎（sherpa-onnx / faster-whisper）识别并自动执行。支持精确整词匹配（置信度 1.0）和子串模糊匹配（置信度 ≤0.95），长触发词优先于短触发词。

---

## 指令详情

### 1. START — 发车/开始作业

| 项目 | 内容 |
|------|------|
| 触发词 | 发车、开车、开始作业、启动、开始干活、start、出发、**走**、**走吧**、**走啊**、**前进**、**开始** |
| 匹配 API | `/api/mission_start` |
| 功能 | 广播 `/chassis/start` 消息，启动小车自动巡航与清除作业模式 |

### 2. STOP — 停车/停止作业

| 项目 | 内容 |
|------|------|
| 触发词 | 停车、收工、停止作业、停下、stop、停止、**停**、**停啦** |
| 匹配 API | `/api/mission_stop` |
| 功能 | 广播 `/chassis/stop` 消息，停止当前巡航与清除作业 |

### 3. ESTOP — 紧急停止

| 项目 | 内容 |
|------|------|
| 触发词 | 急停、紧急停止、立刻停下、紧急停车、estop、紧急、**停停停** |
| 匹配 API | `/api/estop` |
| 功能 | **安全关键！** 立即广播 `/safety_stop` 消息 + 本地立即停止（灭激光、中止点火、命令漏斗复位），无需确认弹窗 |

### 4. CENTER — 云台归中

| 项目 | 内容 |
|------|------|
| 触发词 | 归中、回中、居中、重置、center、**中** |
| 匹配 API | `/api/center` |
| 功能 | 通知点火线程立即中止，舵机云台归中复位到参考位置（SPOT_HOME） |

### 5. LASER_ON — 开启指示激光

| 项目 | 内容 |
|------|------|
| 触发词 | 开激光、打开激光、开指示、开红光、laser on、**激光**、**开灯** |
| 匹配 API | `/api/laser_ir?on=1` |
| 功能 | 打开 S4 红色指示激光（低功率，用于目标指示/瞄准辅助） |

### 6. LASER_OFF — 关闭指示激光

| 项目 | 内容 |
|------|------|
| 触发词 | 关激光、关闭激光、关指示、关红光、laser off、**关掉**、**关灯**、**关闭** |
| 匹配 API | `/api/laser_ir?on=0` |
| 功能 | 关闭 S4 红色指示激光 |

### 7. FIRE_TEST — 试射测试

| 项目 | 内容 |
|------|------|
| 触发词 | 测试、试射、打一下、试一下、点火、fire、**打**、**烧** |
| 匹配 API | `/api/fire_test` |
| 功能 | 执行一次 S3 蓝紫激光短时测试烧灼（默认 0.5s），用于验证激光接口与接线 |

### 8. CLEAR — 开始清除作业

| 项目 | 内容 |
|------|------|
| 触发词 | 清场、开始清除、清除、清理、clear、**清零** |
| 匹配 API | `/planner/start_clearing` |
| 功能 | 通知 strike_planner 决策节点启动目标清除队列（需配合 planner 模块运行） |

### 9. STATS — 查看统计

| 项目 | 内容 |
|------|------|
| 触发词 | 统计、报表、战果、成绩、报告、stats、**结果** |
| 匹配 API | 仅前端显示 |
| 功能 | 在 8093 面板显示当前作业统计信息（打击发数、成功率、命中率等），不触发后端动作 |

## 匹配规则

```
① 精确整词匹配 → 置信度 1.0
   例如: "发车" → 精确匹配 START (1.0)

② 正向子串匹配（触发词 ∈ 识别文本）→ 置信度 ≤0.95
   例如: "请开车" → 含 "开车" → START (0.67)

③ 反向子串匹配（识别文本 ∈ 触发词）→ 置信度 ≤0.85
   例如: "激" → 含在 "开激光" 中 → LASER_ON (0.50)
```

优先级: ① > ② > ③。长触发词（如"紧急停止"）优先于短触发词（如"停止"）。

| RDK端侧 | 小车 RDK X5 | sherpa-onnx Paraformer small | 内置于 8093 |
